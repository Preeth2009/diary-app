# Diary App

A simple Diary App built using Python and File Handling.

## Features

- Write Notes
- Save Notes Permanently
- View Saved Notes
- Exit Program

## Concepts Used

- Functions
- Loops
- Conditional Statements
- File Handling
- User Input
- Menu-Driven Programming

## Menu

1. Write Note
2. View Notes
3. Exit

## Example

### Write Note

Enter Note:
Today I learned file handling.

Note saved successfully.

### View Notes

----- My Notes -----

Today I learned file handling.
Today I completed my Diary App project.

---------------

## Files Used

- diary.txt

The application stores all notes in a text file so they remain available even after the program is closed.

## Technologies Used

- Python
